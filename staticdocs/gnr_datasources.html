Get data sources for the Global Names Resolver.

Usage

gnr_datasources(todf = FALSE, url = "http://resolver.globalnames.org/data_sources.json")

Arguments

todf
Parse id and name of provider to data.frame? logical (default = FALSE).
url
Base url for the API; leave as is.

Value

json or xml output, your choice

Description

Uses the Global Names Index, see http://gni.globalnames.org/ for information.

Examples

## Not run: # # all data sources# gnr_datasources()# # # just id's and names of sources# gnr_datasources(todf=T)# # # give me the id for EOL# out <- gnr_datasources(todf=T)# out[out$title == "EOL", "id"]# # # Fuzzy search for sources with the word zoo# out <- gnr_datasources(todf=T)# out[agrep("zoo", out$title, ignore.case=T), ]# ## End(Not run)
