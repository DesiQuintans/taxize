Query species codes from freshwaterbiology.info

Usage

fresh_codes(x)

Arguments

x
tvt-object; A object of class tvt as returned from fresh_validate.

Value

A data.frame with species Codes (AQEM, DV, TCM, Furse, Perla, Ecoprof).

Description

Query species codes from freshwaterbiology.info

Note

Currently only the macro-invertebrate database is supported.

Examples

## Not run: # spec <- c("Acentrella sinaica",# "Acentria ephemerella",# "Acilius sp.",# "Acroloxus lacustris",# "Allotrichi pallicornis")# # a <- fresh_validate(spec)# fresh_codes(a)# ## End(Not run)
