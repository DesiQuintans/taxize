Get family names to make Phylomatic input object, and output input string to Phylomatic for use in the function phylomatic_tree.

Usage

itis_phymat_format(tsn = NA, format)

Arguments

tsn
quoted tsn number (taxonomic serial number)
format
output format, isubmit (you can paste in to the Phylomatic

Value

e.g., "pinaceae/pinus/pinus_contorta", in Phylomatic submission format.

Description

Get family names to make Phylomatic input object, and output input string to Phylomatic for use in the function phylomatic_tree.

Examples

laply(list("36616", "19322", "183327"), itis_phymat_format, format='rsubmit')
