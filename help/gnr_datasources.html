Get data sources for the Global Names Resolver.

Usage

gnr_datasources(todf = FALSE, url = "http://resolver.globalnames.org/data_sources.json")

Arguments

todf
Parse id and name of provider to data.frame? logical (default = FALSE).
url
Base url for the API; leave as is.

Value

json or xml output, your choice

Description

Uses the Global Names Index, see http://gni.globalnames.org/ for information.

Examples

# all data sources
gnr_datasources()
